                     # Angular Help Commands #

=========================================================================
=> Create a specific version of Angular
npm install -g @angular/cli@11.2.1

-------------------------------------------------------------------------
=> Create an Angular workspace
ng new client

-------------------------------------------------------------------------
=> moving to a client folder
cd client
-------------------------------------------------------------------------
=> run a client
ng serve
-------------------------------------------------------------------------
=> add ngx-bootstrap
Variant 1: npm install ngx-bootstrap --save
Variant 2: ng add ngx-bootstrap // some installing issues
-------------------------------------------------------------------------
=> install specific version of bootstrap
npm install bootstrap@4.6.0
-------------------------------------------------------------------------
=> install font-awesome
npm install font-awesome
-------------------------------------------------------------------------
=> view availible elements to construct
ng g
-------------------------------------------------------------------------
=> view help option for creating a component
ng g c -h
or 
ng g c --help
-------------------------------------------------------------------------
=> creating a nav-bar component
ng g c nav-bar --skip-tests
-------------------------------------------------------------------------
=> moving and creating a core module in the app folder
cd src/app
ng g m core
-------------------------------------------------------------------------
=> creating a shop module
ng g m shop
-------------------------------------------------------------------------
=> --flat puts the file in src/app instead of its own folder. ??
=> --flat doesn't create its own folder
ng g c shop --flat --skip-tests
-------------------------------------------------------------------------
=> creating a service
ng g s shop --skip-tests --tests
-------------------------------------------------------------------------
=> crating a module in app folder
client\src\app> ng g m home
-------------------------------------------------------------------------
=> toaster module
npm install ngx-toastr
